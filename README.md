Trading Simulator (C++)
An interactive command-line trading simulator written in C++ that mimics basic exchange behavior such as order placement, matching, and trade execution.
Features:
Place BUY and SELL orders
Use LIMIT and MARKET order types
Maintain an order book
Cancel open orders
View current order book
View executed trades

Example:
BUY LIMIT RELIANCE 1560 5
SELL LIMIT RELIANCE 1560 2
